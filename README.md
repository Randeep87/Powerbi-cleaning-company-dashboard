
# 📊 Power BI Dashboard – Cleaning Company Analytics

## 📌 Overview

This project presents a professional Power BI dashboard analyzing a cleaning service company’s performance. It provides insights into revenue, profit, employee performance, and service distribution.

## 🎯 Key Features

* KPI Cards: Total Revenue, Profit, Profit Margin,Total Hours worked,Avg Rating
* Top 5 Employees by revenue
* Revenue by Client (Bar Chart)
* Service Type Distribution (Donut Chart)
* Monthly Revenue Trend (Line Chart)
* Conditional Formatting for better insights

## 🛠 Tools Used

* Power BI
* DAX 
* Data Modeling
* Excel (Dataset)

## 📈 Key Insights

* Identified top-performing employees based on revenue
* Highlighted most profitable service types
* Analyzed revenue trends over time

## 📂 Files Included

* Cleaning_Operation_Report.pbix
* Cleaning_Company_Dataset.xlsx

## 🚀 How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Explore interactive visuals
